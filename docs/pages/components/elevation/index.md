---
title: "Elevation"
---

Block elevated by Z axis for 0...24dp
```jsx
{/* Elevation for 4dp */}
<Elevation z={4}>
  {'lift up for 4dp'}
</Elevation>
```

Transition property can be used for smooth elevation from level to level
```jsx
{/* Transition elevation */}
<Elevation 
  transition
  z={4}
>
  {'transition for 4dp'}
</Elevation>
```
